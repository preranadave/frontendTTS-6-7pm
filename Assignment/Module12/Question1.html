What is redux?

Redux is a library of react js. A state management systems for cross-component or app-wide state. Redux is an library that is used to manage data and provides more libraries to manage its data in very simple way. Redux is a useful solution for organising application state and provides a predictable state container for JavaScript application. It's a well-known framework for managing state in React projects, but it also works with Angular, Vue. js, and plain old JavaScript.
